## Auth Audit Target
**this is project to learn vulnerable FastAPI auth target**

Installation & Setup
-
**get the file** 
- git clone git@github.com:Tharatape/auth-audit-target.git
- cd auth-audit-target

**how to install environment**
- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt

**how to run web**
- uvicorn app.main:app --reload

**how to run test**

- for all api test is type command pytest
and you could run vuln2.py and vuln3.py to check vuln 2&3 in proof
